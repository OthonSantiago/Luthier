# Leandro Luthier — site institucional

Nova versão do site da oficina **Leandro Luthier**, especializada em manutenção, revisão e restauração de instrumentos de sopro em Campinas e região.

## Melhorias implementadas

- nova identidade visual responsiva para desktop e celular;
- apresentação direta dos serviços e instrumentos atendidos;
- orçamento pelo WhatsApp como principal caminho de conversão;
- informações de garantia, pagamento, assistência técnica, endereço e horários;
- menu móvel e navegação acessível por teclado;
- SEO local com metadados, dados estruturados, sitemap e robots.txt;
- página 404 personalizada;
- site estático sem dependências ou processo de compilação.

## Executar localmente

```bash
python -m http.server 8080
```

Acesse `http://localhost:8080`.

## Publicação

O projeto pode ser publicado no GitHub Pages, Cloudflare Pages, Netlify ou em qualquer hospedagem de arquivos estáticos.

Antes de substituir o WordPress atual:

1. confirme telefone, endereço, horários, garantia e condições comerciais;
2. copie as imagens utilizadas para a nova hospedagem e ajuste seus caminhos;
3. preserve redirecionamentos das URLs antigas para evitar perda de posicionamento no Google;
4. configure domínio, HTTPS, Google Search Console e Google Business Profile;
5. defina como será mantido o fluxo específico de assistência técnica e envio de nota fiscal.

## Observação sobre as imagens

Esta primeira proposta referencia temporariamente as imagens já publicadas no site atual. Elas devem ser migradas antes que a hospedagem antiga seja desativada.
